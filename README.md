# 🍕 Swiggy SQL Data Analysis

## 📌 Project Overview
Built a full data pipeline to analyze Swiggy food orders using MySQL.

## 🛠️ Tools Used
- MySQL
- Star Schema (Dimensional Modeling)

## 🗂️ Schema Design
- Fact Table: fact_swiggy_orders
- Dimension Tables: dim_date, dim_location, dim_restaurant, dim_category, dim_dish

## 📊 Key KPIs Analyzed
- Total Orders & Revenue
- Monthly & Quarterly Trends
- Top Cities & States by Orders
- Most Popular Cuisines & Dishes
- Customer Spend Distribution
